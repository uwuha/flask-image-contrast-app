# Flask Image Contrast App

![Python CI](https://github.com/uwuha/flask-image-contrast-app/actions/workflows/ci.yml/badge.svg)

Веб-приложение для изменения контраста изображений с построением гистограмм.

## Функциональность

- Изменение контраста изображения
- Построение гистограмм цветов
- Визуализация результатов

## Технологии

- Python 3.8
- Flask
- Pillow (PIL)
- Matplotlib
- GitHub Actions (CI/CD)
